### ✅요구 사항

1. **게시글 작성 기능**
- [x]  게시글 `제목`, `내용`은 필수적으로 포함해야 한다.
- [x]  `제목`은 1글자 이상 15글자 이하여야 한다.
- [x]  `내용`은 1글자 이상 1000글자 이하여야 한다.
- [x]  `제목`은 공백으로만 이루어질 수는 없다.

4. **특정 게시글 수정 기능**
- [x]  게시글 작성할 때의 유효성 검사 조건과 동일하게 가져가야 한다.
