검색과 조회 API에서 `Pageable`을 다루는 부분이 중복된다. *- 메서드로 뺄 수 있을듯?*

### 2. 검색 키워드 기반 검색 기능 리팩토링
- 구리다.. .. .. 그냥 구리다는 말로 밖에 표현이 안된다.
- 검색과 관련된 요청을 `DTO`로 만들어서 API를 개선하고 싶다.
